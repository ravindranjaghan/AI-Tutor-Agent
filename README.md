# My AI Tutor Agent 🤖

This is an AI agent that onboarding users, asks for their name and age, and teaches them topics step-by-step with interactive quizzes!

## How to use it:
Copy this url "https://chatgpt.com/g/g-6a4100926f60819185f89a5ead19fd61-ai-tutor-agent" and paste it into google.

## How to get the best out of it:
First introduce yourself like,"Hi, my name is ____, and I am ___ years old.
Then it will ask you to upload a document or mention the topic you wish to study.
Recommended: Upload a pdf and then specify the topics from that book.

Enjoy learning new things 😊!
